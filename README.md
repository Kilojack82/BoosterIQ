# BoosterIQ

BoosterIQ is an intelligence platform for youth-sports booster clubs. It aggregates the tools clubs already use — Square, Google Drive, SignUp Genius — and produces the decisions volunteer board members actually need: what to buy before the next game, which volunteer slots still need fills, and a printable post-game report. V1 is the concessions and volunteers wedge, piloting with the Lago Vista Vikings Booster.
